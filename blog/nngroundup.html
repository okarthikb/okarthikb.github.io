We have a bunch of points \(\small{(x_1, y_1), \dots, (x_n, y_n)}\), and we need to find a line \(\small{y=\theta_1x+\theta_2}\) that best fits those points, i.e., we need to find \(\small{\theta_1}\) and \(\small{\theta_2}\) that minimises the model error. We need to minimise the sum of the squares of the difference between the actual value and the predicted value for each \(\small{x}\), i.e., we need to find \(\small{\theta_1}\) and \(\small{\theta_2}\) that minimises the following function

\(\small{L(\theta_1, \theta_2)=\sum_{i=1}^n(y_i-(\theta_1x_i+\theta_2))^2}\)
