Regression

We have a dataset \(\small{D}\), i.e., a set of points \(\small{\{(x_i, y_i)\;|\;i=1\dots m\}}\), and we need to find a polynomial (simplest case being a straight line, or linear regression) that best fits the dataset. More formally, we have a polynomial

\(\small{p(x)=\sum\limits_{k=1}^n\theta_kx^k}\)

and we need to find \(\small{\theta}\) that minimises the objective function

\(\small{L(\theta)=\sum\limits_{k=1}^m(y_i-p(x_i))^2}\) \(\small{=\sum\limits_{k=1}^m(y_i-\sum\limits_{k=1}^n\theta_kx_i^k)^2}\)

where \(\small{\theta}\) is the set \(\small{\{\theta_k\;|\;k=1\dots n}\}\) and \(\small{m}\) is the number of data points.
