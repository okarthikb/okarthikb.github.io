Intro

A neural network is simply a function which is tuned to fit a dataset (input-output pairs), i.e., a curve fitter. Linear and polynomial regression are elementary examples of curve fitting - we have data points, and we need to find a curve that fits those points. More formally, we have a dataset \(\small{D}\), a function \(\small{f}\) with parameters \(\small{\theta}\), and we need to find optimal parameters \(\small{\theta^*}\) that minimizes a loss function \(\small{L(D, \theta)}\), where the loss function measures how wrong our model is.

In the case of linear regression, the dataset \(\small{D}\) is a set of \(\small{n}\) pairs of numbers, the function to find is a straight line \(\small{f(x)=mx+c}\), the parameters \(\small{\theta}\) being \(\small{m}\) and \(\small{c}\), and the loss function being

\(\small{L(D, m,c)}\) \(\small{=\sum_{x,y\in D}(y-(mx+c))^2}\)

which we need to minimise by finding \(\small{\theta^*}\), i.e., the right values for \(\small{m}\) and \(\small{c}\). Note that \(\small{x}\) and \(\small{y}\) are constants in the loss function \(\small{L}\). The dataset is constant, the parameters are alterable. How do we find the right values for \(\small{m}\) and \(\small{c}\)? We use a technique called gradient descent.
